# client/command/armory

## Overview

Implements the 'armory' command group for the Sliver client console. Handlers map Cobra invocations to armory workflows such as install, manage, parsers, and search.

## Go Files

- `armory.go`
- `armory_test.go` *(tests)*
- `commands.go`
- `install.go`
- `manage.go`
- `parsers.go`
- `search.go`
- `update.go`
