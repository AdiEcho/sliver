# client/command/pivots

## Overview

Implements the 'pivots' command group for the Sliver client console. Handlers map Cobra invocations to pivots workflows such as details, graph, start, and stop.

- `commands.go`
- `details.go`
- `graph.go`
- `helpers.go`
- `pivots.go`
- `start.go`
- `stop.go`
