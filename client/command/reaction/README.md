# client/command/reaction

## Overview

Implements the 'reaction' command group for the Sliver client console. Handlers map Cobra invocations to reaction workflows such as reload, save, SET, and unset.

## Go Files

- `commands.go`
- `helpers.go`
- `reaction.go`
- `reload.go`
- `save.go`
- `set.go`
- `unset.go`
