# client/command/registry

## Overview

Implements the 'registry' command group for the Sliver client console. Handlers map Cobra invocations to registry workflows such as REG create, REG delete, REG list, and REG read.

## Go Files

- `commands.go`
- `reg-create.go`
- `reg-delete.go`
- `reg-list.go`
- `reg-read.go`
- `reg-write.go`
