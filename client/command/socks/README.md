# client/command/socks

## Overview

Implements the 'socks' command group for the Sliver client console. Handlers map Cobra invocations to socks workflows such as rootcommands, socks start, and socks stop.

## Go Files

- `commands.go`
- `rootCommands.go`
- `socks-start.go`
- `socks-stop.go`
- `socks.go`
