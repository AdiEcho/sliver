# server/cli

## Overview

CLI entrypoint for running the server daemon. Parses flags, configures logging, and launches the service runtime. Key routines cover certs, daemon, operator, and unpack within the cli subsystem.

## Go Files

- `builder.go`
- `certs.go`
- `cli.go`
- `daemon.go`
- `operator.go`
- `unpack.go`
- `version.go`
